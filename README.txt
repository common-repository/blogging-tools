Adds new columns for word count, image count and link counts to posts list.

== Description ==

**Adds 3 new columns to the posts list in the WordPress admin area:**

- Word count
- Image count
- Link counts (internal links and external links)

If you have an idea for a new feature, anything that helps your blogging, please contact the author [here](https://www.bloggingtoolspro.com/). Thank you!

== Installation ==

1. Activate the plugin from WordPress plugin directory or manually upload it to your site
